# 🧠 HCI Neural Quiz

An interactive **Human–Computer Interaction (HCI) quiz application** built with **HTML, CSS, and JavaScript**.
This project provides a visually engaging **neon cyber-style interface** where users can test and improve their knowledge of HCI concepts through a dynamic question bank.

---

## ✨ Features

### 🎮 Interactive Quiz Experience

* Smooth animated transitions between questions
* Real-time progress tracking
* Immediate feedback in Learning Mode
* Final score dashboard with statistics

### 🧩 Multiple Question Banks

Choose the set of questions you want to practice:

* 📚 **Part 1 Only**

  * 279 questions
  * 13 HCI theory sections

* 🔬 **Part 2 Only**

  * 235 technical-term definition questions
  * Focus on key HCI concepts

* ⚡ **All Questions**

  * Combined bank
  * **514 total questions**

---

## 🧠 Quiz Modes

### 🎯 Focus Quiz Mode

* Answer all questions first
* See results only after completing the quiz
* Includes:

  * Score percentage
  * Correct vs wrong answers
  * Detailed review of mistakes

### 📘 Learning Mode

* Instant feedback after each question
* Shows:

  * Whether your answer is correct or wrong
  * Explanation of the correct answer

Perfect for **studying before exams**.

---

## 📊 Results Dashboard

After completing a quiz, the system displays:

* 📈 **Score percentage**
* ✅ Number of correct answers
* ❌ Number of incorrect answers
* 📋 Total questions answered
* 🏷 Grade badge
* 🔎 Review of incorrect answers with explanations

## 🎨 UI & Design Highlights

The interface uses a **futuristic neon theme** inspired by cyberpunk design:

* 🌌 Animated background grid
* ✨ Floating particles
* 🔷 Glassmorphism cards
* ⚡ Neon buttons and glowing borders
* 📡 Holographic scan-line effect

The goal is to make studying **engaging and visually appealing**.
